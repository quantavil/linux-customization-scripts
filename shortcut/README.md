# KDE Plasma Shortcuts Map

## 🪟 Window Management
| Shortcut | Action |
| :--- | :--- |
| **Meta + Q** | Close Window |
| **Meta + M** | Maximize Window |
| **Meta + N** | Minimize Window |
| **Meta + Arrows** | Quick Tile Window (Top/Bottom/Left/Right) |
| **Meta + Shift + Left/Right** | Move Window to Previous/Next Screen |
| **Meta + Alt + T** | Keep Window Above Others |
| **Meta + Ctrl + Esc** | Force Kill Window (Skull Cursor) |
| **Alt + F3** | Window Operations Menu |

## 🚀 Navigation & Workflow
| Shortcut                | Action                                      |
| :---------------------- | :------------------------------------------ |
| **Meta**                | Open Application Launcher                   |
| **Meta + Return**       | Launch Terminal (Konsole)                   |
| **Meta + E**            | Open File Manager (Dolphin)                 |
| **Alt + Space**         | KRunner (Spotlight/Search) *[Rec]*          |
| **Alt + Tab**           | Walk Through Windows                        |
| **Meta + Alt + Arrows** | Focus Window Directionally (Grid Switching) |
| **Meta + W**            | Toggle Overview                             |
| **Meta + G**            | Toggle Grid View                            |
| **Meta + D**            | Peek at Desktop                             |
| **Ctrl + F9**           | Present Windows (Current Desktop)           |
| **Ctrl + F10**          | Present Windows (All Desktops)              |

## 🖥️ Virtual Desktops
| Shortcut | Action |
| :--- | :--- |
| **Meta + Ctrl + Arrows** | Switch One Desktop (Up/Down/Left/Right) |
| **Meta + Ctrl + Shift + Arrows** | Move Window to Desktop (Up/Down/Left/Right) |
| **Meta + F1 / F2** | Switch to Desktop 1 / 2 |

## 🛠️ System & Utilities
| Shortcut | Action |
| :--- | :--- |
| **Meta + I** | Launch System Settings |
| **Ctrl + Shift + Esc** | System Monitor *[Rec]* |
| **Meta + V** | Clipboard History |
| **Meta + L** | Lock Session |
| **Ctrl + Alt + Del** | Log Out Screen |
| **Meta + B** | Switch Power Profile (Battery/Performance) |
| **Meta + 1..9** | Activate Task Manager Entry 1-9 |
| **Meta + Mute** | Mute Microphone |

## 📸 Screenshots (Spectacle)
| Shortcut | Action |
| :--- | :--- |
| **Print** | Capture Full Screen |
| **Meta + Print** | Capture Active Window |
| **Shift + Print** | Capture Rectangular Region |

## 🔍 Accessibility & Zoom
| Shortcut | Action |
| :--- | :--- |
| **Meta + + / -** | Zoom In / Out |
| **Meta + 0** | Zoom to Actual Size |
| **Meta + Ctrl + I** | Invert Colors (Screen) |
| **Meta + Shift + F11** | Clear Mouse Marks |
